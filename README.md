Disha-Setu AI 🚀
The career you never knew existed might be the one you're meant for.

💡 The Problem
In today's world, there are over 500 career paths, yet most students rely on outdated, static, or generic advice. Decision paralysis is at an all-time high, and students often choose paths based on limited awareness rather than genuine potential.

🛠 The Solution
Disha-Setu AI is an intelligent career navigator that bridges the gap between ambition and action. By leveraging advanced Large Language Models, it generates personalized, actionable career roadmaps in seconds—moving beyond generic advice to provide specific "First Moves" and "Hidden Opportunities."

🚀 Key Features
Instant AI Roadmaps: Get a tailored path based on your specific interests (with full support for custom career paths).

Low-Latency Performance: Built using Groq API and Llama 3.1 for sub-second, lightning-fast response times.

Actionable Advice: Every roadmap includes a concrete "First Move" and a "Hidden Opportunity" to gain a competitive edge.

Downloadable Insights: Built-in support to save your career roadmap as a PDF for future reference.

🛠 Tech Stack
Frontend: React.js + Vite

AI Engine: Groq API (Llama 3.1-8b-instant)

Deployment: Vercel

💻 How to Run Locally
Clone the repository.

Create a .env file and add your Groq API Key: VITE_GROQ_API_KEY=your_key_here.

Install dependencies: npm install.

Run the development server: npm run dev.
